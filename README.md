# C# Calculator Application

A simple desktop calculator application developed using C# and .NET Windows Forms.

## Technologies Used

- C#
- .NET Framework
- Windows Forms
- Visual Studio

## Features

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly graphical interface

## Skills Demonstrated

- Object-Oriented Programming (OOP)
- Event-driven programming
- GUI development
- Basic C# application development
